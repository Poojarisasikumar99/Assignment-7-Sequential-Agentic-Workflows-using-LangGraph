
# Assignment 7 — Sequential Agentic Workflows using LangGraph

## Overview

This project demonstrates how to build enterprise-grade sequential AI workflows using:

- LangGraph
- Gemini 2.5 Flash
- Tool Calling
- State Management
- Sequential Node Execution
- LLM-based Decision Making

The assignment contains two real-world use cases:

1. Telecom SIM Activation & Fraud Verification Workflow
2. Healthcare Appointment Prioritization Workflow
